# EQ Auction Forge — dependencies
#
# This app uses the Python standard library plus a single pip package
# (truststore). Install it with:
#     pip install -r requirements.txt
#
# REQUIREMENTS:
#   - Python 3.10 or newer  (truststore requires 3.10+)
#   - tkinter  (GUI toolkit)
#   - truststore  (TLS via the OS-native trust store)
#
# truststore:
#   Makes TLS verification use the operating system's trust store / verifier
#   (Schannel on Windows) instead of OpenSSL's static cert set. This fixes
#   "unable to get local issuer certificate" on lean/older Windows machines
#   whose cert store is missing the API's issuer root — the OS verifier fetches
#   missing roots on demand (the same reason curl/Edge work). It is imported
#   best-effort: on Python <3.10 or if not installed, the app falls back to the
#   default OpenSSL verification (works wherever the OS store already has the
#   root). The shipped Windows .exe bundles it, so end users need nothing.
truststore==0.10.4
#
# About tkinter:
#   tkinter is part of Python's standard library and CANNOT be installed
#   with pip. On Windows and macOS it is included automatically by the
#   official installer from https://www.python.org/downloads/ (make sure
#   "tcl/tk and IDLE" stays checked during install).
#
#   On Linux it may be a separate OS package:
#     Debian/Ubuntu:  sudo apt install python3-tk
#     Fedora:         sudo dnf install python3-tkinter
#     Arch:           sudo pacman -S tk
#
# To run from source:
#     pip install -r requirements.txt
#     python EQ-Auction_Forge.py
