TLP-Auctions API probe diagnostic

Throwaway page to see whether a browser can talk to tlp-auctions.com/api directly. The cert is valid for this host (confirmed), so this is really a CORS test now. Two things to know up front: browsers cannot set a custom User-Agent, and a POST with Content-Type: application/json triggers a CORS preflight (an OPTIONS request). If a call fails, open DevTools → Console/Network — JS only ever sees a generic "Failed to fetch", but the console says whether it was a CORS policy block or a certificate/TLS error.
